DArtE sandbox.

DArtE is a sandbox editor that is developed with and for JMonkeyEngine. The goal is to create a full featured 3D editor that supports all the features of JMonkeyEngine. The current alpha version includes the following possibilities

  • Add j3o files to a scene,
  • Create assemblies from a group of j3o objects
  • An undo/redo framework
  • Live physics interaction
  • Layer system with visibility toggles and locked toggles
  • Procedural animation framework
